## 🚀 Upload to GitHub (2 Methods)

### Method 1: Through GitHub Website (Recommended - 2 minutes)

1. **Open Your Repository**
   - Go to: https://github.com/alihashim786/audio-based-age-prediction

2. **Navigate to Settings**
   - Click the "Settings" tab (⚙️ icon)

3. **Scroll to Social Preview**
   - Scroll down to the section labeled **"Social preview"**
   - It's in the main settings page, about halfway down

4. **Upload Image**
   - Click "Edit" button
   - Click "Upload an image"
   - Select the file: `github-social-preview.png`
   - Wait for upload to complete
   - Click "Save"

5. **Verify**
   - Go back to your repository main page
   - Share the link on any platform (Twitter, LinkedIn, etc.)
   - The banner will appear in the preview!

## ❓ Troubleshooting

**Banner not showing after upload?**
- Wait 5-10 minutes for GitHub to process it
- Clear your browser cache
- Try sharing in a private/incognito window

**Wrong dimensions?**
- The generated image is exactly 1280x640px
- If GitHub rejects it, convert to JPG and try again

**Need to change something?**
- Edit `generate_banner.py`
- Run: `python generate_banner.py`
- Upload the new `github-social-preview.png`
